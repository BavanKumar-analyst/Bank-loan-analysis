# Power BI Dashboard – Bank Loan Analysis

This Power BI report was built using data extracted from a MySQL database.
It includes:

- KPI cards for loan applications, funded amount, and amount received
- Month-to-Date (MTD) and Month-over-Month (MoM) trend analysis
- Good vs Bad loan performance
- Loan status–level drill-down and filtering

The dashboard is designed to support business decision-making and executive reporting.
